class

        - ES6라고 불리는 ECMAScript 2015부터 class라는 문법이 추가되었다.       
        - class는 constructor와 prototype을 합친 문법이다.
        - 클래스는 객체를 생성하기 위한 템플릿이다.
        - 생성자를 가지고 있다 java에서는 클래스명과 동일하게 클래스명 One 생성자 또한 One이 된다
        - 그런데 javascript에서는 생성자를 constructor로 사용합니다
        - what? 생성자는 무엇인가요? 초기 속성을 지정하는게 
        - 클래스는 객체가 아닌 템플릿이다
        - 그럼 상속을 받나요?
          상속을 받으면 객체로 정의하고 물려받는데 => 빈 껍데기만 있으면 사용
          그렇지 않으면 객체정의 하고 속성 정의하고 사용한다
